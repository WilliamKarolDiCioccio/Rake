# RAKE ENGINE

#### Description

Rake will be a fully featured game engine with custom rendering, physics and audio systems. The project aims to give everyone the possibility to create his virtual world and tell the story he wants, every kind of game will be realizable, 2D & 3D, third & first person, parkour, shooters and turn-based games. Rake will provide the creator with plenty of easy-to-use tools to speed up and simplify the work giving him the possibility to focus on the artistic part of the work. It would be fantastic if the program also became an educational platform to teach all interested parties, with simplicity but without trivializing, the technologies behind video games.



A small note on the name, which is not exactly self explaning. Rake gives the project an aggressive look, referring the thought to the urban legend of the Rake, moreover, the term rasterization derives from rake, which, for the less informed is the rendering technique currently most in use.



*"Life is to short to write an engine for each game"*

(Taken from the Book *3D Graphics Programming* by Sergei Savchenko)
